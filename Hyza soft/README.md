# HYZA-SOFT - Sistema de Registro

Sistema simples para registro de produtos com controle de devolução. Interface nas cores creme e azul marinho com marca d'água "HYZA-SOFT" ao fundo.

## Funcionalidades
- ✅ Barra de pesquisa integrada
- ✅ Adição de registros com Data, Produto, Lote, Nome e Devolução
- ✅ Lista organizada por data (decrescente)
- ✅ Exclusão automática quando "Devolução = Sim"
- ✅ Armazenamento local (localStorage)
- ✅ Design responsivo

## Como usar
1. Preencha os campos no formulário
2. Clique em "ADICIONAR INFORMAÇÃO"
3. Use a barra de pesquisa para filtrar
4. Clique no "✕" para remover um item

- Destaques: Azul Marinho (#1f2c48)
